# KwikSnax Redux — Vision

## Short Description

**KwikSnax Redux** is a modern remake and spiritual successor to the original *KwikSnax*, designed as a fast-paced arcade puzzle-action game for handheld gaming consoles.

The game focuses on precise, responsive, and predictable gameplay inspired by classic arcade titles of the late 80s and handheld console games of the 90s.
The project is developed with real hardware execution as a primary target, respecting the technical limitations of the intended platforms.

KwikSnax Redux also serves as a showcase project for the [ToyGine2](https://github.com/ToymanInteractive/toygine2) engine, demonstrating its capabilities when developing games for resource-constrained systems.

## Project Goals

- Recreate the feel of classic arcade gameplay with highly responsive controls
- Achieve minimal input latency on real hardware
- Implement fully deterministic game simulation
- Ensure stable performance on limited handheld hardware
- Support cooperative gameplay
- Make the game suitable for speed-running
- Use the project as a technical showcase for the ToyGine2 engine
- Respect the constraints and behavior of real console hardware
- Provide high replay value through simple but deep mechanics

## Unique Features

- Fully deterministic gameplay
- Minimal input latency
- Designed for real hardware, not emulator-first
- Support for handheld gaming consoles
- Support for dual-screen configurations
- Cooperative multiplayer mode
- Short play sessions with fast restart
- Mechanics designed for mastery and optimization
- Used as a demonstration project for the ToyGine2 engine

## Project Tone

The game should feel like a classic handheld arcade title:

- fast and responsive gameplay
- cozy retro atmosphere
- light cartoon-style presentation
- clean and readable visuals
- precise controls
- fair and predictable mechanics
- authentic late-90s console feel

The design combines a friendly visual style with strict, skill-based gameplay.

## Target Audience

- Players who enjoy classic arcade games
- Retro console enthusiasts
- Players who prefer short play sessions
- Players interested in cooperative gameplay
- Speed-runners and skill-focused players
- Developers interested in low-level console development
- Users playing on real hardware (GBA / NDS / compatible systems)
