# KwikSnax Redux — Core Loop

## Игровая сессия

Игровая сессия представляет собой последовательность уровней, которые игрок проходит до поражения или завершения игры.

Типичная сессия включает:

1. Запуск игры
2. Выбор режима (одиночная игра / кооператив)
3. Начало уровня
4. Прохождение уровня
5. Переход к следующему уровню при успехе
6. Потеря жизни при ошибке
7. Завершение игры при потере всех жизней или выполнении условий победы
8. Возможность немедленного перезапуска

Сессии должны быть короткими и повторяемыми.
Игрок должен иметь возможность быстро начать новую попытку без длительных загрузок или пауз.

## Основной цикл

Основной игровой цикл строится вокруг повторяющейся последовательности действий игрока:

1. Анализ текущей ситуации на уровне
2. Перемещение персонажа по игровому полю
3. Взаимодействие с объектами уровня
4. Избегание врагов и опасностей
5. Уничтожение врагов или выполнение условий уровня
6. Очистка уровня
7. Переход к следующему уровню

Во время игры игрок постоянно принимает быстрые тактические решения:

- куда двигаться
- когда атаковать
- когда избегать опасности
- как эффективнее очистить уровень
- как минимизировать риск

Основной цикл является:

- быстрым
- читаемым
- предсказуемым
- пригодным для повторения много раз подряд

## Прогресс

Прогресс в игре происходит в рамках одной игровой сессии.

Основные элементы прогресса:

- последовательное прохождение уровней
- постепенное увеличение сложности
- увеличение количества врагов
- изменение конфигурации уровней
- ускорение игрового темпа
- рост требований к точности игрока

Дополнительно прогресс может включать:

- подсчёт очков
- бонусы за быстрое прохождение
- бонусы за эффективную игру
- таблицу рекордов

Прогресс не усложняет базовые правила игры.
Усложнение происходит за счёт скорости, количества объектов и взаимодействия механик, а не за счёт введения новых сложных систем.
