# KwikSnax Redux — Controls

## Базовая схема

- Игрок использует стандартную схему кнопок портативной консоли (GBA / NDS).
- Основные элементы управления:

| Кнопка          | Действие                                                         |
| --------------- | ---------------------------------------------------------------- |
| D-pad / стрелки | Перемещение персонажа (вверх, вниз, влево, вправо)               |
| A               | Основное действие (сбор/атака/взаимодействие с объектом)         |
| B               | Отмена / вторичное действие / уклонение                          |
| L / R           | Дополнительные функции / переключение режимов (по необходимости) |
| Start           | Пауза, меню                                                      |
| Select          | Быстрый доступ к статистике / информации уровня                  |

- Для портативных систем с двумя экранами управление адаптируется так, чтобы основной экран оставался зоной действия, а дополнительный экран отображал информацию и подсказки без отвлечения игрока.

## Действия

Основные действия игрока:

- **Перемещение** — движение персонажа по уровню в четырёх направлениях.
- **Сбор предметов / взаимодействие** — активация предметов, сбор бонусов, уничтожение врагов.
- **Уклонение / вторичное действие** — безопасное перемещение, прыжок или уклонение от опасностей.
- **Меню и пауза** — управление игровым состоянием, просмотр статистики, перезапуск уровня.

Дополнительные действия являются контекстными. Доступное действие определяется ближайшим интерактивным объектом или текущим состоянием игрока.

## Контекстные правила

- Каждая кнопка выполняет действие только в соответствующем контексте.
- Пример: кнопка A атакует врага только если игрок находится рядом, иначе собирает предмет.
- Контекстные действия не блокируют основной игровой цикл: игрок всегда может перемещаться и уклоняться.
- В кооперативном режиме контекстные действия учитывают положение и состояние всех игроков для предотвращения конфликтов.

## Доступность

- Управление использует минимальный набор кнопок. Каждая кнопка выполняет одно основное действие или одно контекстное действие.
- Альтернативные схемы управления поддерживаются, включая схему для левшей.
- Поддержка вибрации (Rumble) для обратной связи:
  - короткая вибрация при сборе предмета
  - средняя вибрация при контакте с врагом или ловушкой
  - длительная вибрация при потере жизни или завершении уровня
- Вибрация настраивается в меню, чтобы игрок мог включать, отключать или регулировать интенсивность.
